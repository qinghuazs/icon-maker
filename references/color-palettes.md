## 设计系统参考

需要更系统的配色时：

- **Tailwind**：https://tailwindcss.com/docs/customizing-colors — 11 种色系，每种 11 级（50-950）
- **Material 3**：https://m3.material.io/styles/color/system/overview — Primary / Secondary / Tertiary
- **Apple HIG**：https://developer.apple.com/design/human-interface-guidelines/color — iOS/macOS 系统色
- **Ant Design**：https://ant.design/docs/spec/colors-cn — 中文友好，国内风格

## 色名解析提示

用户描述模糊时按以下匹配：

- "紫色" / "purple" → `#8B5CF6`
- "深紫" / "靛蓝" → `#5B50E8`（默认）
- "蓝色" / "blue" → `#3B82F6`
- "红色" / "red" → `#EF4444`
- "绿色" / "green" → `#10B981`
- "黄色" / "yellow" → `#F59E0B`
- "黑色" / "black" → `#111827`
- "白色" / "white" → `#FFFFFF`
- "灰色" / "gray" → `#6B7280`
- "渐变"（无具体色） → 品牌紫渐变 `#6366F1 → #8B5CF6`

## 颜色对比建议

避免低对比度组合：
- ❌ 浅灰背景 + 白图标
- ❌ 黄色背景 + 白图标
- ✅ 深色背景 + 白图标（推荐）
- ✅ 浅色背景 + 深色图标

默认配色对比度足够（深紫 #5B50E8 + 白）。
